Breast Cancer Classifier

How it works

This is a breast cancer classifier, it uses Convolutional Neural Networks to analyse the different views of the mammography screening. It outputs the probability of belonging to a BI-RADS class (0, 1 or 2).

BI-RADS CLASSIFICATION

  • BI-RADS 0: Additional imaging evaluation and/or comparison to prior mammograms is needed.
  • BI-RADS 1: Negative. There’s no significant abnormality to report.
  • BI-RADS 2: This is also a negative mammogram result (there’s no sign of cancer), but the radiologist chooses to describe a finding known to be benign, such as benign calcifications, lymph nodes in the breast, or calcified fibroadenomas.

    • Model architecture

    The model computes the output in two stages. In the first stage, a number of convolutional and pooling layers is separately applied to each of the views. View specific representations are concatenated to form a vector. The vector is the input to the fully connected layer followed by a softmax to output the probability of belonging to each BI-RADS class.
